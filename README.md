# Escrity-lenguaje
Este es un intento de crear un lenguaje de programacion sencillo si mucha cosa lo que tendra es  lo basico de cualquier lenguaje como la declaracion de variables , funcion para imprimir texto, para evaluar expresiones matematicas, bucles, condicionas ya lgunas cosas propias. Este lenguaje estara escrito en python hasta que se cambie de opinion(lo cual se ve dudoso y dificil pero dado el caso se eligiria el lenguaje C o C#) esta siendo escrito en tiempos libres y se plantea poderlo usar en el programa charDeveloping. Es un proyecto de tiempos libres y se lo hace por oseo.
